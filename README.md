# 18歲前必懂的民法知識 MVP

一個可展示的靜態 Web App，協助高中生用生活情境理解成年前後常見民法概念。

## 功能
- 首頁與進度摘要
- 6 張知識卡片
- 5 題情境測驗
- 結果頁
- 手機版 RWD
- LocalStorage 保存學習進度與測驗分數

## 使用方式
直接開啟 `index.html`，或用本機靜態伺服器：

```bash
python -m http.server 8000
```

然後開啟 http://localhost:8000

## 資料來源
內容參考法務部《民法成年指南》相關主題，已改寫成高中生容易理解的語氣。此 MVP 為法治教育展示用，不構成法律意見。

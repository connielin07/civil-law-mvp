# 18歲前必懂的民法知識 MVP Guide

## 目標
製作一個可展示的 MVP Web App，主題為「18歲前必懂的民法知識」。內容參考法務部《民法成年指南》，但以高中生能理解的生活語氣重寫，不直接複製原文。每個知識點標註指南中的來源主題。

## 技術架構
- 單頁式靜態 Web App
- HTML + CSS + Vanilla JavaScript
- 不需要後端與資料庫
- 使用 LocalStorage 保存進度
- 可直接用瀏覽器開啟，也可用 GitHub Pages / Netlify / Vercel 部署

## 頁面與功能

### 1. 首頁
- Hero 標題：「18歲前必懂的民法知識」
- 副標：用生活語氣說明成年前後要知道的契約、金錢、打工、租屋與賠償責任
- 顯示學習進度：完成卡片數、測驗狀態、最高分
- CTA：開始學習、直接測驗

### 2. 6 個知識卡片
每張卡包含：標題、標籤、高中生語氣說明、生活例子、一句重點提醒、來源主題、「我懂了」按鈕。

卡片主題：
1. 18歲成年是什麼意思？
2. 簽契約不是只是簽名
3. 父母還能不能幫我決定？
4. 打工與金錢責任
5. 租屋、分期、網購要注意什麼？
6. 做錯事可能要賠償

### 3. 情境測驗
- 5 題情境選擇題
- 每題 3 個選項
- 答題後顯示簡短說明與來源主題
- 完成後進入結果區

### 4. 結果頁
- 顯示答對題數
- 顯示學習進度
- 依分數給不同鼓勵與建議
- 提供重新測驗、回到知識卡、清除進度

資料格式：

```json
  "completedCards": [1, 2, 3],
  "quizCompleted": true,
  "quizScore": 4,
  "lastCompletedAt": "2026-07-02T10:00:00.000Z"
}
```

## RWD 設計
- Mobile first
- 手機版單欄卡片
- 平板以上改為雙欄
- 桌機版知識卡片最多三欄
- 底部固定導覽：首頁 / 知識 / 測驗
- 按鈕與選項加大，方便觸控

## 內容原則
- 不使用艱深法律術語，必要時用生活例子解釋
- 不提供個案法律意見，只做基本法治教育
- 每個重點標註「法務部《民法成年指南》：來源主題」
- 若之後取得正式 PDF 頁碼，可把主題標註補成頁碼
